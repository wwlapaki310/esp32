# esp32
* M5stack(esp32）で温度、加速度、ジャイロを測定＆ディスプレイ上に可視化
* SoftAPモードを使い、M5stackをWifiルータと設定
* M5stackから温度データをchart.jsでグラフ化したhtmlを配信するサーバとして設定
